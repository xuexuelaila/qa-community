# 🎬 百问百答社区系统 - 演示指南

## 快速体验

### 1. 确认服务运行

```bash
cd /Users/fangyaxin/qa-community
./start.sh
```

等待几秒后，访问：http://localhost:3000

### 2. 航海日志页面演示

**访问**: http://localhost:3000

#### 功能演示：

1. **日期导航**
   - 左右滑动查看不同日期
   - 点击日期切换
   - 当前日期高亮显示

2. **搜索功能**
   - 在搜索框输入关键词（如"Next.js"）
   - 实时显示搜索建议
   - 点击标签进行筛选

3. **今日概览**
   - 查看AI生成的每日总结
   - 统计数据展示
   - 点击"收起/展开"按钮

4. **知识卡片**
   - 查看问题和答案
   - 点击标签查看分类
   - 点击"查看原始对话"展开详情
   - 点击👍或👎进行反馈

### 3. 求助站页面演示

**访问**: http://localhost:3000/community

#### 功能演示：

1. **状态分类**
   - 点击"全部/待解决/已解决"切换
   - 查看不同状态的帖子

2. **发起求助**
   - 点击"+ 发起求助"按钮
   - 填写问题标题
   - 选择当前阶段
   - 描述具体卡点
   - 说明已做尝试
   - 上传附件（可选）
   - 点击"发布求助"

3. **浏览帖子**
   - 查看帖子列表
   - 查看用户信息和角色标识
   - 查看回复数和浏览数
   - 点击卡片查看详情

### 4. API接口测试

#### 测试Q&A接口

```bash
# 获取每日Q&A列表
curl http://localhost:3001/api/qa/daily

# 搜索Q&A
curl "http://localhost:3001/api/qa/search?keyword=Next.js"

# 获取每日概览
curl http://localhost:3001/api/qa/brief
```

#### 测试Posts接口

```bash
# 获取帖子列表
curl http://localhost:3001/api/posts

# 获取待解决的帖子
curl "http://localhost:3001/api/posts?status=pending"

# 健康检查
curl http://localhost:3001/health
```

## 常见问题

### Q: 为什么有些功能点击没反应？

A: 当前使用Mock数据，部分交互功能（如发布帖子、回复等）只是前端演示，不会真正保存数据。连接数据库后即可正常使用。

### Q: 如何修改Mock数据？

A: 编辑 `backend/src/routes/mock.ts` 文件，修改 `mockQAs` 和 `mockPosts` 数组。

### Q: 如何添加新的演示数据？

A: 在Mock数据数组中添加新对象，遵循现有的数据结构即可。
