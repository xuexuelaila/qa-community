# 抖音CPS项目知识沉淀

## 今日知识概览

本文档整理了抖音三期2群中关于抖音CPS（Cost Per Sale）项目的核心知识点，涵盖账号养号、视频制作、流量优化、违规词规避、密令口令使用、佣金结算等实操问题。内容来源于群内教练、助教与学员的真实问答交流。

## 精选问答

### Q1: 账号养号与流量测试
**标签:** #账号运营 #流量测试 #养号

**问题:** 新注册的抖音账号需要如何养号？如何判断账号是否正常？

**回答:**
1. **养号流程:**
   - 新注册账号后，先发布一条10秒以上的随拍视频（非业务内容）
   - 观察6小时内自然流量是否达到50以上
   - 如果随拍视频6小时内播放量＜50，说明账号存在异常，建议更换账号
   - 如果随拍播放量＞50，账号正常，可以开始发布业务内容

2. **注意事项:**
   - 注销号后重新注册，建议先发非业务视频测试自然流量
   - 连续3条不同内容6小时内播放均不到50，建议更换账号或换平台发布（如B站、视频号）
   - 账号名字和头像短期内频繁修改可能影响权重

**来源:** 教练-海宇、助教-念安、助教-程程

### Q2: 密令词与口令的区别及使用
**标签:** #密令 #口令 #推广链接

**问题:** 密令词和口令有什么区别？如何使用？锁单时间是多久？

**回答:**
1. **密令与口令的区别:**
   - 密令词：由平台分配，每个人专属，有效期较长（如到2月28号）
   - 口令：自己在万单宝生成，有效期30天，快到期需要更换

2. **锁单规则:**
   - 用户通过口令/密令进入会场后，12小时内购买会场已有类目的商品算你的佣金
   - 如果购买的是非会场类目商品，则不计入
   - 密令和口令的锁单时间都是12小时

3. **使用建议:**
   - 可以口令一个会场、密令一个会场，组合推广
   - 新手建议集中一个会场，口令和密令使用同一个会场
   - 口令放在标题首行，用户点击转发后可自动复制

**来源:** 助教-雯雯、助教-程程、飞龙胡公子

### Q3: 视频违规词规避
**标签:** #违规词 #敏感词 #限流

**问题:** 抖音CPS视频中有哪些违规词需要规避？如何替换？

**回答:**
1. **常见违规词及替换:**
   - "京东" → 东哥、某东、东东、□东、🐶东
   - "红包" → 包子
   - "优惠券" → 消费券、福利、专属权益
   - "搜索" → 输入、打字、敲字
   - "国家补贴/政府补贴" → 国补（简称可用）

2. **图片中的违规词:**
   - 截图中出现"领券中心""大额优惠""京东"等字样需要打马赛克
   - 建议对所有可能触发敏感词的文字进行遮挡处理
   - 规避一切可能被限流的风险

3. **检测工具:**
   - 零克查词、句易网、轻抖app
   - 可以使用抖+（DOU+）投流测试内容是否违规

**来源:** 助教-念安、助教-程程、航小助

### Q4: 视频播放量低的排查方法
**标签:** #播放量 #限流 #内容优化

**问题:** 视频发布后播放量很低（个位数或十几个），如何排查原因？

**回答:**
1. **排查流程:**
   - 先确认账号是否正常：发随拍视频测试，6小时内是否破50
   - 使用DOU+测试：如果审核通过说明内容符合商业流量规则，但自然流量低需要优化内容
   - 检查违规词：文案、字幕、画面、配音中是否含有敏感词
   - 检查图片：是否有违规文字，建议逐个测试图片

2. **内容优化方向:**
   - 前3秒要精心设计，吸引用户停留
   - 视频倍速调整为1.2/1.3，节奏要快
   - 添加确定性信息（如订单截图）
   - 注意音频配合，配适配的BGM
   - 避免同质化过高，修改脚本增加差异化

3. **测试方法:**
   - 把图片全部撤掉测试流量，判断是否图片违规
   - 投自定义DOU+测试，看是否能通过审核
   - 连续发3条不同内容测试，判断是账号问题还是内容问题

**来源:** 教练-海宇、助教-念安、KAI-成都-Lv.1

### Q5: 剪映视频制作技巧
**标签:** #剪映 #视频制作 #曼波配音

**问题:** 使用剪映制作视频时有哪些注意事项？如何设置导出参数？

**回答:**
1. **导出参数设置:**
   - 分辨率：选择"智能分辨率"或1080p
   - 帧率：60fps
   - 格式：MP4
   - 建议勾选"一键超清"

2. **视频规格:**
   - 横屏（16:9）：建议曼波形式
   - 竖屏（9:16）：建议口播形式

3. **曼波配音:**
   - 必剪的曼波音效比剪映的更有情绪波动
   - 文本读错需要重新生成该部分音频
   - 小猪客服特效：抖音拍摄视频→选择特效→搜索"猪猪"

4. **破解版注意:**
   - 下载后第一件事在设置里关掉自动更新
   - 不要登录账号
   - 正版先关闭，再打开破解版

**来源:** 助教-程程、助教-念安、宗-桂林-Lv1

### Q6: 万单宝平台使用
**标签:** #万单宝 #推广链接 #密令申请

**问题:** 如何在万单宝平台生成推广链接和查看密令？

**回答:**
1. **生成口令:**
   - 登录万单宝平台
   - 点击【推广活动】
   - 选择想要推广的活动
   - 点击【生成链接】
   - 选择推广位
   - 点击【生成推广链接】，即可生成口令

2. **查看密令:**
   - 进入万单宝→我的密令
   - 查看"密令信息"（不是"主题信息"）
   - 密令有效期可在平台查看（如到2月28号）

3. **推广位说明:**
   - 推广位只是记录数据的位置，不影响佣金
   - 忘记复制口令可以重新生成

4. **密令申请:**
   - 注册万单宝平台
   - 在登记表单提交手机号激活权限
   - 目前航海只分配一个家电会场密令

**来源:** 助教-程程、助教-楚楚、助教-媛媛

### Q7: 视频标题与话题设置
**标签:** #标题 #话题 #关键词

**问题:** 视频发布时标题和话题应该如何设置？

**回答:**
1. **标题设置:**
   - 口令放在标题最前方（首行），用户点击转发后自动复制
   - 标题要体现视频主要信息，踩中关键词
   - 不要出现平台词（如美团、京东）
   - 标题空着，口令前面不要放任何内容

2. **话题标签:**
   - 产品词：苹果、3C数码、家电等
   - 活动词：国补、年货节等
   - 攻略词：攻略、购物攻略等
   - 话题中可以出现#美团 #京东，不算违规
   - 不要堆砌标签，选1-2个相关话题即可

3. **简介文案:**
   - 前20字必须抓眼球
   - 说明视频能解决什么问题
   - 引导点赞、评论或点击购物车

**来源:** 航小助、助教-雯雯、助教-程程

### Q8: 佣金结算与订单跟踪
**标签:** #佣金 #订单 #结算

**问题:** 如何查看订单和佣金？佣金什么时候结算？

**回答:**
1. **订单查看:**
   - 在万单宝平台查看订单数据
   - 自己下单测试可以在订单中查到
   - 外卖订单需要等待一段时间才能显示

2. **佣金规则:**
   - 用户通过你的链接进入会场，12小时内购买会场类目商品算你的佣金
   - 如果用户先用你的口令进入，又搜索别人的密令，算最新打开的那个入口
   - 佣金按月结算

3. **GMV排行:**
   - 可以在万单宝平台查看GMV排行
   - 上榜门槛每天都在提升（第1天279元，第2天2070元）

**来源:** 助教-程程、助教-楚楚、教练-海宇

### Q9: 多平台分发策略
**标签:** #多平台 #B站 #快手

**问题:** 除了抖音，还可以在哪些平台发布CPS视频？

**回答:**
1. **推荐平台:**
   - B站：可以同步发布
   - 视频号：可以尝试
   - 快手：流量可以，但转化率较低

2. **发布建议:**
   - 同一个视频可以在抖音发布后，当天下午发到B站
   - 如果抖音连续3条视频6小时播放不到50，建议换平台
   - 快手播放量可能很高（几千），但转化率低，评论多但不下单

3. **注意事项:**
   - 不同平台的推荐机制不同
   - 小红书不推荐做CPS
   - 视频号需要认证出镜人，抖音目前不需要

**来源:** 助教-程程、助教-楚楚、猛人初一

## 实操技巧总结

### 内容创作五要素（怡然老师分享）

1. **节奏:** 前3秒精心设计，3秒后倍速1.2/1.3，单刀直入
2. **观感:** 添加动效、字幕加粗、综艺效果
3. **确定性信息:** 订单截图在开头和主体内容中反复出现
4. **情绪刺激:** 自己看完视频，检查情绪是否被调动
5. **音频配合:** 配适配的BGM，前3秒人声加扩音器特效

### 评论区运营

- 用小号在评论区贴买到的截图
- @豆包提问"真的假的"，提升信任
- 可以将口令放在评论区，增加引导入口
- 用别的号在评论区互动引导

### 账号安全设置

- 可以关闭"可能认识的人""通讯录好友""地理位置推荐"
- 不影响视频流量
- 发视频不需要实名认证（直播需要）

## 常见错误避免

1. **不要使用对标视频的密令词**，必须换成自己的
2. **不要在图片中保留"京东""优惠券""领券中心"等字样**
3. **不要频繁修改账号名字和头像**
4. **不要使用极限词**（第一、顶尖、最佳、独家等）
5. **不要在标题口令前放任何内容**
6. **不要使用破解版剪映后自动更新**
7. **不要连续发同质化内容**
